On Tuesday we released the latest version of the PLOS Article-Level Metrics application (https://web.archive.org/web/20170731155505/http://article-level-metrics.plos.org/?). As always, the source code is available at Github (https://web.archive.org/web/20170731155505/https://github.com/articlemetrics?). The changes in this version focus on improving API performance, making it easier to install the application, and RSS feeds for the most popular articles by source and publication date (e.g. the most tweeted papers published in the last 7 days (https://web.archive.org/web/20170731155505/http://alm.plos.org/sources/twitter.rss?days=7?)). See the Github Wiki page (https://web.archive.org/web/20170731155505/https://github.com/articlemetrics/alm/wiki/2.6?)for more details, in the Wiki you also find the development roadmap (https://web.archive.org/web/20170731155505/https://github.com/articlemetrics/alm/wiki/Roadmap?) and the issue tracker (https://web.archive.org/web/20170731155505/https://github.com/articlemetrics/alm/issues?) for feature suggestions.
