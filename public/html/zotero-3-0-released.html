Zotero 3.0 was officially released (https://www.zotero.org/blog/zotero-3-0-is-here/?) today. The big change in version 3.0 of the reference manager is a standalone version that runs outside the Firefox browser. The first beta (https://blogs.plos.org/mfenner/2011/08/23/zotero-3-0-beta-released-works-with-chrome-and-safari/?) was released in August 2011.
